
INPUT SCAN - SISTEMA DE ESCANEO
===============================

Versión: 2025.09.29
Fecha: 2025-09-29

DESCRIPCIÓN:
Input Scan es un sistema profesional de escaneo para entornos de manufactura.

REQUISITOS DEL SISTEMA:
- Windows 10/11 (64-bit)
- 100 MB de espacio libre en disco
- Conexión a red (para base de datos MySQL)

INSTALACIÓN:
1. Ejecute el archivo de instalación como Administrador
2. Siga las instrucciones del asistente
3. La aplicación se instalará en Archivos de Programa
4. Se creará un acceso directo en el escritorio

PRIMER USO:
1. Configure la conexión a la base de datos en el archivo .env
2. Ejecute la aplicación desde el menú Inicio o escritorio
3. Use sus credenciales para iniciar sesión

SOPORTE:
Para soporte técnico contacte al administrador del sistema.
